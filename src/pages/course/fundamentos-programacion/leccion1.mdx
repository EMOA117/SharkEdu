# Introducción a las Variables

En esta lección, aprenderás qué son las variables y cómo se utilizan en programación.

## ¿Qué es una variable?

Una variable es un contenedor para almacenar un valor que puede cambiar durante la ejecución de un programa.

```python
x = 5
y = "Hola, Mundo!"
print(x, y)
```

En el ejemplo anterior, `x` es una variable que almacena un número entero (`int`) y `y` es una variable que almacena una cadena de texto (`str`).
